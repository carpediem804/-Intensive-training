시작하기

사진에 자동으로 해쉬태그와 제목 달아주기

얼굴상, 감정, 첫인상, 나이, 성별 해쉬태그와 해쉬태그 기반으로 제목 달아주기

Automatic hash tag, Automatic name

시스템의 특징

얼굴상, 첫인상 해쉬태그

딥러닝(VGG 11)을 이용하여 얼굴상(고양이상,강아지상,공룡상,조류상,말상,여우상,개구리상)과 첫인상(감정이과한,과묵한,불만많은,사연있는,호감형,화끈한) 해쉬태그

감정, 나이, 성별 해쉬태그

Microsoft FACE API를 이용하여 감정(화남,경멸,역겨움,두려움,행복,무표정,슬픔,놀람) 나이(유아기,10대,20대,30~40대,50대), 성별(남,여) 해쉬태그

자동 제목 완성

머신러닝(KNN) 기반으로 해쉬태그와 관련있는 사진의 제목 달아주기
